Le crate digger qui creuse trois fois.

Telecharger pour Windows

Double-clic, aucune installation. Pas besoin de Python ni de ffmpeg.

C'est quoi ?

DDD nettoie ta bibliothèque musicale et la passe en qualité jouable en club, tout seul. Il démasque les faux lossless (des MP3 déguisés en .flac/.wav), va chercher une vraie version sur Soulseek (FLAC, WAV ou AIFF, avec repli MP3 320 automatique), vérifie chaque fichier au spectre, et le range dans une bibliothèque propre. Pas besoin d'être développeur : c'est une fenêtre.

1 · DIG

Tes sources

Scanne n'importe quel dossier de ton disque, ou récupère tes favoris Discogs et Bandcamp.

2 · DOWNLOAD

Soulseek

Cherche FLAC / WAV / AIFF, avec repli MP3 320 automatique, un statut live par piste et reprise automatique.

3 · DETECT

Le spectre

Analyse FFT : classe en Lossless / HQ / Douteux / Mauvais, + contrôle durée et identité. Seul le bon morceau, au-dessus de ton seuil, est gardé.

À quoi ça ressemble

Deux onglets, une fenêtre native. Pas de terminal.

Le truc qui change tout : le spectre fait loi

Un MP3 320 réencodé en FLAC reste un MP3. Les filtres de Soulseek ne le voient pas : le conteneur affiche fièrement « 1411 kbps ». DDD, lui, regarde le spectre réel : un mur de fréquences à 16 kHz trahit la source lossy. Le format et le bitrate déclarés ne servent qu'à la recherche Soulseek, jamais à la décision de garder ou rejeter. Le spectre ne ment pas, les tags si : c'est ce qui distingue un vrai 320 / lossless d'un upscale. Chaque fichier téléchargé est ré-audité (spectre + durée + identité titre/artiste) et classé par son cutoff réel. Tu choisis le seuil minimum à garder via trois presets : DJ Club (>= 18 kHz, défaut, MP3 320 inclus), Audiophile (>= 20 kHz) ou Puriste (lossless pur). Les MP3 sous 320 kbps sont bannis quoi qu'il arrive ; le reste part à la corbeille.

Comment ça marche

Trois clics dans la fenêtre.

  1. Renseigne tes accès (Réglages) : login Soulseek, token Discogs, et le dossier bibliothèque.
  2. Scanne un dossier (ou récupère tes favoris) : DDD classe chaque fichier et repère les doublons.
  3. Upgrade : ce qui passe ton seuil atterrit dans ta bibliothèque, le reste part à la corbeille. Une seule bibliothèque propre, dédoublonnée.

Aussi dispo en ligne de commande : ddd scan, ddd upgrade, ddd import, ddd scrape, ddd gui. Coeur Python portable (Windows / Mac / Linux).
