C'est quoi ?

Maintenir une bibliothèque DJ en vrai lossless à la main, c'est des heures perdues : vérifier la source de chaque track, retracker les fichiers foireux, refaire la même recherche pour la 4e fois. DDD le fait à ta place.

1 · DIG

Tes sources

Scrape tes favoris Discogs et Bandcamp, ou scanne n'importe quel dossier déjà sur ton disque.

2 · DIG

Soulseek

Va chercher le vrai lossless en match strict, avec retry automatique sur les ratés.

3 · DIG

Le spectre

Analyse FFT de chaque fichier pour démasquer les faux lossless (un MP3 déguisé en FLAC/WAV).

Le truc qui change tout : l'anti-upscale

Un MP3 320 réencodé en FLAC reste un MP3. Les filtres de Soulseek ne le voient pas - le conteneur affiche fièrement « 1411 kbps ». DDD, lui, regarde le spectre réel : un mur de fréquences à 16 kHz trahit la source lossy. Chaque fichier téléchargé est ré-audité, et n'entre dans ta bibliothèque que s'il est vraiment lossless.

✓ Vrai lossless → gardé ~ Suspect (320k) → revue ✗ Faux lossless → rejeté

Comment ça marche

Trois clics dans la fenêtre, ou en ligne de commande.

  1. Choisis un dossier de musique (ou colle ta wishlist).
  2. Scanne : DDD classe chaque fichier (vrai lossless, faux, lossy) et repère les doublons.
  3. Upgrade : pour les fichiers flaggés, il télécharge le vrai lossless et remplace - seulement si l'audio passe le contrôle.

Aussi dispo en CLI : ddd scan, ddd upgrade, ddd scrape, ddd gui. Coeur Python portable (Windows / Mac / Linux).
