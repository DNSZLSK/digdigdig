Le crate digger qui creuse trois fois.

Dig tes sources  ->  Dig Soulseek  ->  Dig le spectre

Telecharger pour Windows

Double-clic, aucune installation. Pas besoin de Python ni de ffmpeg.

C'est quoi ?

DDD nettoie ta bibliothèque musicale et la passe en vrai lossless, tout seul. Il démasque les faux FLAC (des MP3 déguisés), va chercher la vraie version sur Soulseek, vérifie au spectre qu'elle est authentique, et la range dans une bibliothèque propre. Pas besoin d'être développeur : c'est une fenêtre.

1 · DIG

Tes sources

Scanne n'importe quel dossier de ton disque, ou récupère tes favoris Discogs et Bandcamp.

2 · DOWNLOAD

Soulseek

Va chercher le vrai lossless, avec un statut live par piste et reprise automatique.

3 · DETECT

Le spectre

Analyse FFT + contrôle de la durée et de l'identité : seul le bon morceau, vraiment lossless, est gardé.

À quoi ça ressemble

Deux onglets, une fenêtre native. Pas de terminal.

Le truc qui change tout : l'anti-upscale

Un MP3 320 réencodé en FLAC reste un MP3. Les filtres de Soulseek ne le voient pas : le conteneur affiche fièrement « 1411 kbps ». DDD, lui, regarde le spectre réel : un mur de fréquences à 16 kHz trahit la source lossy. Chaque fichier téléchargé est ré-audité (spectre + durée + identité titre/artiste), et n'entre dans ta bibliothèque que s'il est vraiment le bon morceau en lossless. Le reste part à la corbeille.

Vrai lossless -> bibliotheque Suspect (320k) -> revue Faux / mauvais match -> corbeille

Comment ça marche

Trois clics dans la fenêtre.

  1. Renseigne tes accès (Réglages) : login Soulseek, token Discogs, et le dossier bibliothèque.
  2. Scanne un dossier (ou récupère tes favoris) : DDD classe chaque fichier et repère les doublons.
  3. Upgrade : les vrais lossless atterrissent dans ta bibliothèque, les faux partent à la corbeille. Une seule bibliothèque propre, dédoublonnée.

Aussi dispo en ligne de commande : ddd scan, ddd upgrade, ddd import, ddd scrape, ddd gui. Coeur Python portable (Windows / Mac / Linux).
